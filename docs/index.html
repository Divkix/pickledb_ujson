Welcome

pickleDB-uJson is a lightweight and simple key-value store. This version is built using the ujson module to improve performance. It is built upon ujson module and was inspired by redis. It is licensed with the BSD three-clause license.

pickleDB is Fun

>>> import pickledb_ujson as pickledb

>>> db = pickledb.load('example.db', False)

>>> db.set('key', 'value')
True

>>> db.get('key')
'value'

>>> db.dump()
True

And Easy to Install

$ pip install pickledb-ujson

More Information

You can view all of pickleDB's commands and what they do here.

pickleDB was written by Harrison Erd and this is a modified version by Divanshu Chauhan and uses uJson module instead of the standard json module provided by Python. If you would like to file an issue report or fork the project, check out the Github project page. You can also take a look at pickleDB on PyPI.

pickleDB got its name from Python's "pickle" module, which it previously used. However, now pickleDB uses the "json" module. It is faster and cleaner, but this version is using 'uJson' module, which is even faster.
